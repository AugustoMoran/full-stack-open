# Bloglist Frontend — Full Stack Open Parte 5

Frontend para la aplicación Bloglist (parte 5 del curso).

## 🚀 Desarrollo local

1) Instalar dependencias:

```bash
npm install
```

2) Configurar el backend:

Define `VITE_API_URL` en `.env` (por defecto `http://localhost:3003`).

3) Ejecutar:

```bash
npm run dev
```

## ✅ Funcionalidades
- Login con JWT
- CRUD de blogs
- Likes y borrado
- UI con componentes reutilizables

## 🧪 Tests

```bash
npm run test:run
```

## 🧪 E2E (Playwright)

```bash
npm run test:e2e
```
